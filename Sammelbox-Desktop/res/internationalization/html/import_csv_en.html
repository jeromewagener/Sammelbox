Importing a CSV file

Sammelbox allows you import CSV (comma seperated value) files as new albums. Such files can be created using Spreadsheet applications such as Micosoft Excel or Open Office Calc. The content of these files is usually of tabular nature whereby each column is seperated by a comma or a similar symbol.

Example:

Book Title;Author;Pages
Book 1;Author 1;42
Book 2;Author 2;21
Book 3;Author 3;88

Specifying an image column

Sammelbox allows to import images for each entry. The images must be referenced by their complete path. (e.g. C:/My-Pictures/picture1.jpg) If multiple images should be imported, then these images must be separated by a predefined character.

Specifying field types

Sammelbox will import by default all columns (fields) as simple text. However, you can add a specific fieldtype to a header column in order to manually define the filetype. (Must be appended after the '~' character)

Example:

Book Title~TEXT;Author~TEXT;Pages~INTEGER

Available field types and accepted values

The following table will list the available field types and accepted field values:

Type Accepted Values
TEXT Almost everything. However, please note that text fields have limited search capabilities.
DECIMAL Decimal numbers (e.g: 42.21)
INTEGER Integers (e.g: 21)
DATE A date in the format specified by the settings. (E.g: 16/01/1988) Please verify your settings if you are unsure about the current format!
URL A web link (E.g: www.sammelbox.org)
OPTION A yes, no & unknown field. Accepted values are 'YES', 'NO' and 'UNKNOWN'
STAR_RATING A star rating between 0 and 5. Accepted values are 'ZERO_STARS', 'ONE_STAR', 'TWO_STARS' 'THREE_STARS', 'FOUR_STARS' and 'FIVE_STARS'.

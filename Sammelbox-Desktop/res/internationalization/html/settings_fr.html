Settings

The sidepane allows you to modify your settings. The table below describes the different options.

Setting Description
Language The language to be used. By default, the system language will be used. Please note that changing the language will require Sammelbox to be restarted.
Default View The view that is used by default to show albums. If the selected view is not applicable to the current album, then the detailed view will be used by default.
Date Format The date format used to show date fields. This format will also be used to read/write dates when importing/exporting
Full Synchronization When synchronising with a mobile device, Sammelbox synchronizes by default the original images (full). Using this option, it possible to only synchronize the thumbnails, reducing the space required on your mobile device.
